Welcome!

This website offers the possibility of executing and visualizing subgroup algorithms step by step. For now, the possibilities are:

CN2-SD

This algorithm consists on a modification of the classification rule learner algorithm CN2, adapted to subgroup discovery, with extra features like weighted relative accuracy. It is based on an article that can be found in here. The step by step execution of this algorithm lets you observe the data in a dynamic scatterplot whose axes will be selected taking into account the rules that are being found (that describe subgroups). A slight modification of VizRank is used to choose the best two axes (for class separation) in the first visualization. Also, the points in the graph will vary in size, depending to the weight of each point (used in the algorithm). To start the execution of this algorithm with your own data, click on the next button:
Start!

PRIM

This algorithm is the patient rule induction method and the article in which the implementation is based can be found here. It is designed to find (greedily) boxes that characterise different subgroups in the data. These boxes are built by an iterative top-down peeling process, a bottom-up pasting technique and a check for redundant variables. The step execution of this algorithm lets you observe the data in a dynamic scatterplot whose axes will be selected taking into account the variables selected to peel in the top-down process. Also, a slight modification of VizRank is used to choose the best two axes (for class separation) in the first visualization of each step. To start the execution of this algorithm with your own data, click on the next button:
Start!
